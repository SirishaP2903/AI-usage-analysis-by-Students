# AI Impact on Student Life — Tableau Dashboard (2026)

An interactive Tableau dashboard exploring how AI tool adoption is shaping academic performance, productivity, and career confidence among university students across six disciplines.

---

## Overview

This project analyses a dataset of **1,500 students** surveyed in 2026 on their use of AI tools in academic life. The dashboard surfaces patterns across GPA changes, time savings, usage behaviours, and ethical attitudes — broken down by major, AI tool, and use case.

The goal is to move beyond surface-level adoption statistics and answer a more meaningful question: **is AI actually helping students, and where?**

---

## Dataset

**File:** `AI_Impact_Student_Life_2026.csv`

| Field | Description |
|---|---|
| `Student_ID` | Unique student identifier |
| `Age` | Student age (18–25) |
| `Major` | Field of study |
| `Primary_AI_Tool` | Main AI tool used |
| `Task_Frequency_Daily` | Average daily AI interactions |
| `Main_Usage_Case` | Primary academic use case |
| `GPA_Baseline` | GPA before AI adoption |
| `GPA_Post_AI` | GPA after AI adoption |
| `Time_Saved_Hours_Weekly` | Estimated weekly time savings |
| `AI_Ethics_Concern` | Perceived ethical concern (Low / Medium / High) |
| `Career_Confidence_Score` | Self-rated career confidence (1–10) |
